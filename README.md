# Selenium Automation Framework

## Overview
This project is a Selenium Automation Framework built using Python, Selenium WebDriver, and Pytest. It automates the core user workflows of the SauceDemo web application using the Page Object Model (POM).

## Features
- Login Automation
- Inventory Testing
- Cart Testing
- Checkout Testing
- Page Object Model (POM)
- Explicit Waits
- Logging
- Configuration Management
- GitHub Actions (CI/CD)

## Tech Stack
- Python
- Selenium WebDriver
- Pytest
- WebDriver Manager
- Git
- GitHub
- GitHub Actions
